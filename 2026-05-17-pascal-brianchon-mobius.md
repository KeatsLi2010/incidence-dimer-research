## 第一部分：莫比乌斯 球面模型

### 基本原理

将平面 $ \mathbb{R}^2 $ 通过球极投影嵌入单位球面 $ S^2 \subset \mathbb{R}^3 $：

$$(x,y) \mapsto \left(\frac{2x}{x^2+y^2+1}, \frac{2y}{x^2+y^2+1}, \frac{x^2+y^2-1}{x^2+y^2+1}\right)$$

在射影 3-空间 $ \mathbb{P}^3 $ 中的关键对应：

| 平面 $ \mathbb{R}^2 $ 中的对象 | $ \mathbb{P}^3 $ 中的对象 |
|---|---|
| 点 | 二次曲面 $ Q $ 上的点 |
| 圆 | $ Q $ 与某平面的交线 |
| 直线 | $ Q $ 与过北极 $ N $ 的平面的交线 |
| 四点共圆 | 四点共面（在 $ Q $ 上）|

### 验证结果

用 Python 随机生成 100 组配置，帕斯卡 定理通过率 99/100，布列安桑 通过率 100/100。

## 第二部分：帕斯卡 定理在 莫比乌斯 模型中的形态

### 原定理

> 六边形内接于一圆锥曲线，则三对对边的交点共线。

### 莫比乌斯 升维后的形态

1. 六个点 $ A_1, \ldots, A_6 $ 在圆上 → 六个点 $ A_1', \ldots, A_6' $ 在 $ Q \cap \sigma $ 上（共面）
2. 边 $ A_iA_{i+1} $ → 过 $ N, A_i', A_{i+1}' $ 的平面 $ \alpha_{i,i+1} $
3. 对边交点 $ X = A_1A_2 \cap A_4A_5 $ → 两平面 $ \alpha_{12} \cap \alpha_{45} $ 与 $ Q $ 的交点 $ X' $
4. 帕斯卡 结论 "$ X,Y,Z $ 共线" → $ N, X', Y', Z' $ 共面

**数值验证**: $ \det(N, X', Y', Z') = 5.33 \times 10^{-18} \approx 0 $

## 第三部分：布列安桑 定理（对偶）

### 原定理

> 六边形外切于一圆锥曲线，则三对对顶点的连线共点。

### 莫比乌斯 升维后的形态

布列安桑 是 帕斯卡 关于 $ Q $ 的极对偶：

- 切点 $ T_i $ → $ T_i' \in Q $
- 切线 → $ T_i' $ 处 $ Q $ 的切平面，方程为 $ T_i' \cdot x = 1 $
- 布列安桑 共点 $ B $ → $ B' \in Q $，其极平面包含三条对顶点连线

## 第四部分：二聚体模型 编码 —— 关键发现

### 核心困难

$ \mathbb{P}^3 $ 中，回路 的大小上界为 4（共面四点，无三点共线）。6 个共圆的点**不构成 回路**，因为它们在 $ \mathbb{P}^3 $ 中不是极小线性相关的。

这意味着：

> **圆条件不能编码为 回路 条件 (V)，必须编码为 相干性 条件 (F)**。

### 编码方案

帕斯卡 定理的标准 梅涅劳斯 证明正好提供了编码：

1. 取 $ AB, CD, EF $ 构成的三角形 $ \triangle $
2. 对 $ \triangle $ 应用 **三次 梅涅劳斯 定理**：
   - 第一次：过 $ BC $ 的截线
   - 第二次：过 $ DE $ 的截线
   - 第三次：过 $ FA $ 的截线
3. 圆锥曲线的**交比恒等式**作为 相干性 条件
4. 三式相乘 → 得到 帕斯卡 线条件

### 在 二聚体模型 中的对应

```
        每个 梅涅劳斯 应用  ←→  一个 相干的拼图面
        交比恒等式         ←→  面的 多重比 = 1
        三式乘积 = 1       ←→  球面拼图的切除性质
```

这恰好对应 **2505.02229 论文中的球面三角剖分主定理**：
- 三个 梅涅劳斯 三角形 = 球面的三角剖分的三个面（像 德萨格 的四顶点四面体拼图）
- 相干性 条件来自圆锥曲线（不是来自一般位置假设）
- 球面上的"切除"自动给出 帕斯卡 线

**结构对应**：

| 德萨格 定理的拼图证明 | 帕斯卡 定理的拼图证明 |
|---|---|
| 4 个三角形的四面体拼图 | 3 个三角形的 梅涅劳斯 × 3 |
| 3 个面 相干性 → 第 4 个面 相干性 | 圆锥曲线条件 + 2 个 梅涅劳斯 → 第 3 个面 相干性 |
| 结论 = 第四个面的 相干性 = 德萨格 线 | 结论 = 第三个 梅涅劳斯 = 帕斯卡 线 |

### 一般化

**猜想**: 任何涉及单个圆锥曲线和有限个点的关联定理，只要能用 梅涅劳斯 + 圆锥曲线交比恒等式证明，都可以编码为 二聚体模型 框架中的球面拼图。

这包括：
- 帕斯卡 定理（六点）
- 布列安桑 定理（对偶，六条切线）
- 蝴蝶定理（四点共圆 + 一条弦）
- 米克尔 定理（四个圆 → 反演 → 线性 + 一个圆条件）

### 对于 布列安桑

布列安桑 的情况更微妙。在 莫比乌斯 模型中，布列安桑 的对偶性体现为：
- 6 条切线 → 6 个切平面（在 $ Q $ 上）
- 布列安桑 共点 → 极平面上的条件

这暗示 布列安桑 在 $ \mathbb{P}^3 $ 中的编码需要用到**对偶 dimer 图**（交换黑白顶点），与极对偶关于 $ Q $ 自然对应。

## 第五部分：扩展边界

### 可以处理的

| 定理类型 | 编码方式 | dimer 图结构 |
|---------|---------|------------|
| 帕斯卡 (n 点在圆锥曲线上) | 梅涅劳斯 × 3 + 交比 相干性 | 球面，3 面 |
| 布列安桑 (n 条切线) | 极对偶 + 帕斯卡 | 球面，3 面（对偶图）|
| 单圆 + 多点关联 | 反演 → 直线 → dimer | 取决于定理 |
| 米克尔 型（四圆）| 反演到线性 + 单圆 相干性 | 环面变体 |

### 不能直接处理的

- 涉及多个圆锥曲线且无法用一个反演统一消去的定理
- 需要度量（角度、长度）的定理
- 高次曲线（三次及以上）

### 关于椭圆等非圆圆锥曲线

任何非退化圆锥曲线都射影等价于一个圆。所以处理流程是：
1. 射影变换把圆锥曲线变为圆
2. 圆通过 莫比乌斯 升维到 $ \mathbb{P}^3 $
3. 在 $ \mathbb{P}^3 $ 中应用 dimer 框架
4. 射影变换回原曲线（射影变换保持所有关联关系）

## 结论

1. **莫比乌斯 升维是连通"圆世界"和"二聚体模型 线性框架"的桥梁**。

2. **圆条件不能是 回路 (V)，必须是 相干性 (F)**。$ \mathbb{P}^3 $ 中 回路 最多 4 个点。

3. **帕斯卡 定理恰好对应一个 3 面的球面拼图**——与 德萨格 定理的 4 面四面体拼图是同一框架的不同实例。

4. **布列安桑 通过对偶自动跟随**。

5. **这是 二聚体模型 框架首次被用来理解含圆锥曲线的定理**，尽管框架本身是纯线性的。
