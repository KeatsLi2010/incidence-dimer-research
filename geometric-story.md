# 拼图的故事 —— 用几何语言重述关联几何与圆

## 一、什么是"拼图"？

想象你手里有一张**橡皮膜**——它可以是球面、环面（甜甜圈），或者更复杂的曲面。
你在这张膜上画了一张**二部图**：顶点被交替染成白色和黑色。

现在给每个顶点贴上几何标签：

- **白顶点** → 一个点在射影空间中
- **黑顶点** → 一条线（或高维的超平面）

图的每一个面被边界上的顶点围成一个多边形，由于顶点黑白交替，每个面有偶数条边。

这整张橡皮膜连同上边的图和几何标签，就叫一张**拼图**（tiling）。

```
        白顶点(点)         黑顶点(线)
           ○                 ●
          /|\               /|\
         / | \             / | \
        ●  ○  ●           ○  ●  ○
       /|  |  |\         /|  |  |\
      
      图的一个面（2n边形，顶点黑白交替）
           ○
          / \
         ●   ●     ← 点-线-点-线... 围成一圈
          \ /
           ○
```

## 二、拼图的两条规则

一张拼图要被称作"好的"（即**相干双回路构型**），必须满足两条规则：

### 规则一：回路条件（V）

围绕每个顶点的邻居，必须构成一个**回路**（circuit）——也就是"最小线性相关"。

在平面上：
- 围绕一个**黑顶点**（它代表一条线），它的白顶点邻居们必须**共线**——它们都落在这条线上。3 个点共线 = 回路。
- 围绕一个**白顶点**（它代表一个点），它的黑顶点邻居们必须**共点**——这些线都穿过这个点。3 条线共点 = 回路。

```
   回路(黑顶点)：邻居白点共线        回路(白顶点)：邻居黑线共点
   
        ○₁                              ●₁
        |                               /
        ● —— ○₂ —— ○₃                 ○ —— ●₂
        |                               \
        ○₄                              ●₃
   
   四点共线 = 回路(在ℙ²中)           三线共点 = 回路(在ℙ²中)
```

**几何直觉**：回路就是"恰好不多不少地拴在一起"。太少（两个对象）就"自由"，太多就"过约束"。回路是**最小的非平凡关系**。

### 规则二：相干条件（F）

每个拼图面，把它的顶点按顺时针方向取多重比，结果必须等于 1。

$$\frac{\ell_1(A_1) \cdot \ell_2(A_2) \cdots \ell_n(A_n)}{\ell_1(A_2) \cdot \ell_2(A_3) \cdots \ell_n(A_1)} = 1$$

在平面上，最简单的四边形拼图面 $A, c, B, d$（A,B 是点，c,d 是线）：

```
         线c
    A ●──────────● B
      │          │
      │   拼图面  │
      │          │
      └──────────┘
         线d

   相干条件 = 1  ⟺  线AB、线c、线d 三线共点
```

**几何直觉**：相干条件是"面与面之间的拼合无矛盾"——所有面的多重比都是 1，就像齿轮完美啮合。

## 三、主定理：拼图的魔力

> **拼图主定理**：如果闭合曲面上一张拼图的所有面**除一个外**都是相干的，那么最后一个面也自动相干。

```
    球面上的拼图：
    
        面1 ✓        面2 ✓
       ┌────┐       ┌────┐
       │    │       │    │
       └────┘       └────┘
        面3 ✓    →   面4 自动✓  ← 主定理说的就是这个！
```

这就是拼图框架的**证明引擎**。你把已知的关联条件分给 $n-1$ 个面，主定理自动推出结论——也就是最后一个面的相干性。

**为什么能自动成立？**

本质是：每个面的多重比 = 1 是一个代数方程。当你把球面上所有面拼在一起，面与面之间共享顶点。相邻面的多重比互相消去，最后只剩最后一个面的多重比——由于所有其他面的多重比都是 1，最后一个面的多重比也只能是 1。

这就是一个**全局拓扑约束**（球面是单连通的）产生**局部代数恒等式**的过程。

## 四、城区改造：拼图的变形术

拼图有两个"合法变形"——做这些操作后，拼图依然满足回路+相干条件。

```
  城区改造（Urban Renewal）：
  
  改造前：                改造后：
      ○                     ○
     /|\                   /|\
    ● | ●       →         ●─○─●
     \|/                   \|/
      ○                     ○
  
  正方形面变成对角线交叉，点和线重新安置。
  但——拼图的"整体关联结构"保持不变！
```

**城区改造 + 恢复组合结构 = 离散可积系统**。

这就是它叫"动态关联几何"的原因：你在一张图上做一系列城区改造，最终图回到原来的形状（组合结构不变），但几何数据变了。新的几何数据自动满足旧的关联条件——你得到了一个**自动定理生成器**。

## 五、Pascal 定理的拼图证明

### 原定理

> 六边形 $A_1A_2A_3A_4A_5A_6$ 内接于一圆锥曲线。
> 则三对对边的交点 $X = A_1A_2 \cap A_4A_5$、$Y = A_2A_3 \cap A_5A_6$、$Z = A_3A_4 \cap A_6A_1$ **共线**。

```
         A₁
        /  \
       /    \
   A₆ ●      ● A₂
      |       |
      |  圆   |
   A₅ ●      ● A₃
       \    /
        \  /
         A₄
         
   对边交点：X = A₁A₂∩A₄A₅
            Y = A₂A₃∩A₅A₆    →  X,Y,Z 共线 = Pascal线
            Z = A₃A₄∩A₆A₁
```

### 拼图视角

**第一步**：取三角形 $\triangle = A_1A_2 \cap A_3A_4 \cap A_5A_6$（由交替的三条边张成）。

**第二步**：在这个三角形上使用**三个 Menelaus 拼图面**：

```
             顶点U
             /|\
            / | \
    A₁A₂ ──/──●──\── A₁A₆
          /   |   \
         /    |    \
    A₂A₃●─────●─────●A₅A₆
         \    |    /
          \   |   /
    A₃A₄ ──\──●──/── A₄A₅
            \ | /
             \|/
             顶点V

    三个 Menelaus 面 = 球面上的 3 面拼图
```

- 面 1：截线 $A_2A_3$ 穿过三角形 → Menelaus 条件 = 多重比 = 1
- 面 2：截线 $A_4A_5$ 穿过三角形 → Menelaus 条件 = 多重比 = 1
- 面 3：截线 $A_6A_1$ 穿过三角形 → 通过圆锥曲线的交比恒等式，面 3 也是相干的

**第三步**：由**拼图主定理**，三个面的相干性互相约束，乘积 = 1 推出 $X,Y,Z$ 共线。

### 与 Desargues 定理的对比

| | Desargues | Pascal |
|---|---|---|
| 拼图曲面 | 球面 | 球面 |
| 面的数量 | 4（四面体）| 3（三角形剖分）|
| 相干条件的来源 | 一般位置 | 圆锥曲线的交比恒等式 |
| 结论 | Desargues 线 | Pascal 线 |

两者是**同一框架的不同实例**！

## 六、Möbius 升维：圆如何在拼图中变成线

这是整个研究的核心洞见。

### 问题

拼图框架只能处理**线性**对象（点、线、面...平行宇宙）。圆是**二次**的，不能直接放进拼图。

### 方案：给平面充气变成球

取一个单位球面 $S^2$。把平面通过**球极投影**贴到球面上：

```
        北极 N
         ●
        /|\
       / | \          球极投影：
      /  |  \         (x,y) → (2x, 2y, x²+y²-1) / (x²+y²+1)
     /   ○   \        ─────────────────────────────
    /  ／ ＼  \       平面上的点 → 球面上的点
   ●──●     ●──●      平面上的圆 → 球面上一个平面截出的圆
  /    ＼ ／    \     平面上的线 → 球面上过北极的平面截出的圆
 ●              ●
```

在射影 3-空间 $\mathbb{P}^3$ 中，球面 $S^2$ 是一个**二次曲面** $Q$。

| 在平面上 | 在 $\mathbb{P}^3$ 中 |
|---|---|
| 平面上的一个点 | 二次曲面 $Q$ 上的一个点 |
| 平面上的一个圆 | $Q$ 与某个平面的**交线** |
| 平面上的共圆四点 | **共面的四点**（在 $Q$ 的限制下）|

### 关键突破

> 在 $\mathbb{P}^3$ 中，4 个点构成**回路** ⟺ 它们共面。
> 在 $Q$ 的限制下，"四点共面" = "四点共圆"。

但注意——6 个共圆点不构成回路！在 $\mathbb{P}^3$ 中回路最多 4 点。

所以**圆条件不是回路条件 (V)，而是相干条件 (F)** —— 它通过圆锥曲线的**交比恒等式**进入拼图框架。

### Pascal 在升维后

```
  平面上的 Pascal：              ℙ³ 中的 Pascal：
  
  6点在圆上                      6点在Q上，共面(都在平面σ上)
  对边相交 → 3个交点             每边过北极N → 是Q上的大圆
  3交点共线 = Pascal线           3交点+N共面 = Pascal线平面
```

## 七、Brianchon：镜像世界

Brianchon 是 Pascal 关于二次曲面 $Q$ 的**极对偶**。

```
  在 Q 上：
  一个点 p 的极平面 = 过 p 的切平面
  一条线 ℓ 的极点 = 线与 Q 的两个交点的极线交点
  
  Pascal（点世界）：         Brianchon（线世界/极对偶世界）：
  6点在圆上                 6条线切于圆
  对边交点共线              对顶点连线共点
```

在拼图框架中：把 Pascal 拼图的所有白/黑顶点互换（点 ↔ 线），就得到 Brianchon 的拼图。

## 八、总结

1. **拼图 = 二部图 + 几何标签 + 回路条件 + 相干条件**
2. **主定理 = 球面上 n-1 个面相干 ⇒ 第 n 个面相干**
3. **城区改造 = 保持相干的局部操作 = 离散可积系统**
4. **Möbius 升维 = 圆 → 二次曲面上的线性配置**
5. **圆条件 = 相干条件 (F)，不是回路条件 (V)**
6. **Pascal = 球面上的 3 面拼图，Brianchon = 它的镜像**
